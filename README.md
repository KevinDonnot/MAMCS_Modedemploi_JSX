Script ExtendScriptToolkit utilisé pour la production du catalogue de l'exposition Mode d'emploi au Musée d'art moderne et contemporain de Strasbourg (2024).

Génère les 800 exemplaires uniques de la publication en numérotant chaque exemplaires et en intégrant, pour chaque artiste et chaque génération, une visualisation dédiée spécifique (fichiers Tiff bitmap non inclus dans le repo).
